# Demo Script

## 3-Minute Flow

1. Open AdverSim on the dashboard.
2. Navigate to **Adversary Builder**.
3. Select **Credential Compromise Chain**.
4. Set:
   - Target user: `finance.admin`
   - Target host: `NYC-WKS-014`
   - Intensity: `Medium`
   - Duration: `30 minutes`
   - Noise level: `Realistic`
5. Click **Run Adversary Simulation**.
6. Review generated synthetic telemetry.
7. Open **Detections** and show the correlated findings.
8. Open **Timeline** and explain the reconstructed chain.
9. Open **Reports** and show the analyst-ready Markdown report.

## Presentation Line

AdverSim's Adversary Builder lets defenders safely stage realistic threat behavior, generate synthetic telemetry, and practice detection and response without touching real systems.
